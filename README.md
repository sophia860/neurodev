# Neurodev

> **Code like your brain was always right.**

Neurodev is a Whop-native product brand selling neuroscience-powered software tools that help developers code faster, focus deeper, and think clearer.

Built for the developer who always knew something was different about how their brain works — and is finally ready to use that as an advantage.

## What is Neurodev?

The average dev tool was built for the neurotypical majority. Neurodev builds tools shaped around how developer brains *actually* work — accounting for hyperfocus, context-switching cost, dopamine prediction loops, and non-linear thinking patterns.

The neuroscience is real. The tools are tested on the person who built them first.

---

## Tools

| Tool | Cognitive pattern | What it does |
|------|------------------|--------------|
| **FlowLock** | FLOW | Detects your cognitive state and shields your context window from interruption |
| **ContextStack** | CONTEXT | Snapshots working memory so switching tasks costs seconds, not minutes |
| **DeepDial** | FOCUS | Calibrates ambient conditions to your dopamine rhythm |
| **TraceMap** | MEMORY | Externalises working memory as a navigable graph |

---

## The Science

- Dopamine isn't motivation. It's prediction.
- Flow state has a neurological signature. We track it.
- Context switching costs 23 minutes. Our tools account for it.
- The ND brain doesn't lack focus. It allocates it differently.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local`:
   ```
   GEMINI_API_KEY=your_key_here
   ```
3. Run the app:
   ```bash
   npm run dev
   ```
